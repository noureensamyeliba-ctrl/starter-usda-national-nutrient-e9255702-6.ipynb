📊 Exploratory Data Analysis (EDA) Notebook

This repository contains a Python notebook for performing basic exploratory data analysis on train.csv and test.csv datasets.

🚀 Overview

The notebook helps you understand the structure and behavior of the data through visualization and summary statistics. It focuses on identifying patterns, distributions, and relationships between features before building any machine learning model.

📂 Dataset
train.csv – Training dataset used for analysis and modeling preparation
test.csv – Test dataset used for structure comparison and validation
🛠️ Tools & Libraries
Python
Pandas
NumPy
Matplotlib
Scikit-learn
📈 Visualizations Included
Column-wise distribution plots (histograms & bar charts)
Correlation matrix heatmap
Scatter matrix with density plots
⚙️ Key Functions
plotPerColumnDistribution() – Visualizes feature distributions
plotCorrelationMatrix() – Shows relationships between numerical features
plotScatterMatrix() – Displays pairwise relationships and density plots
🎯 Goal

To gain initial insights into the dataset, detect patterns, and support feature understanding for future machine learning tasks.
